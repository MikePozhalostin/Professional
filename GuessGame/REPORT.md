# Отчёт по SOLID

## Принцип единственной ответственности
Каждый класс выполняет одну задачу:
- GuessGame — логика игры
- RandomNumberGenerator — генерация числа
- ConsoleInput / ConsoleOutput — ввод и вывод
- NumberComparer - Сравнение чисел

## Принцип открытости/закрытости
Игра расширяется через интерфейсы INumberGenerator, INumberComparer без изменения существующего кода.

## Принцип подстановки Барбары Лисков;
Любая реализация INumberGenerator может быть подставлена в GuessGame без изменения поведения.

## Принцип разделения интерфейса;
Интерфейсы разделены: IUserInput, IUserOutput, INumberGenerator, INumberComparer.

## Принцип инверсии зависимостей;
GuessGame зависит от абстракций, а не от конкретных реализаций.

## Время выполнения
Примерно 2 - 2.5 часа
